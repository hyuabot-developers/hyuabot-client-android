# Native/UI 기능 테스트 시나리오

## 앱 바로가기
- 런처에서 HYUabot 앱 아이콘을 길게 누른다.
- `셔틀`, `학식`, `열람실`, `지도` 바로가기가 표시되는지 확인한다.
- `셔틀` 바로가기를 누르고 셔틀버스 실시간 화면이 열리는지 확인한다.
- `학식` 바로가기를 누르고 학식 화면이 중식 탭으로 열리는지 확인한다.
- `열람실` 바로가기를 누르고 열람실 좌석 현황 화면이 열리는지 확인한다.
- `지도` 바로가기를 누르고 캠퍼스 지도 화면이 열리는지 확인한다.

## 기능 안내 다시 보기
- `설정` 화면을 연다.
- `기능 안내 다시 보기` 항목을 누른다.
- 완료 토스트가 표시되는지 확인한다.
- 셔틀버스, 버스, 학식, 열람실, 지도, 설정 등 코치마크가 있는 화면을 다시 연다.
- 앱을 삭제하지 않아도 코치마크가 다시 표시되는지 확인한다.

## 교내 연락처 액션
- `메뉴 > 교내 연락처` 화면을 연다.
- 연락처 행 하나를 길게 누른다.
- `전화번호 복사`를 선택한다.
- 다른 입력창에 붙여넣어 복사된 전화번호가 올바른지 확인한다.
- 같은 연락처 또는 다른 연락처 행을 다시 길게 누른다.
- `연락처 공유`를 선택한다.
- 시스템 공유 시트가 열리고 연락처 이름과 전화번호가 공유 내용에 포함되는지 확인한다.
- 연락처 행을 일반 탭한다.
- 전화 앱이 열리고 해당 전화번호가 입력되어 있는지 확인한다.

## 학식 운영 상태 표시
- `학식` 화면을 연다.
- 조식, 중식, 석식 탭을 전환한다.
- 각 식당의 보조 설명에 운영 시간과 운영 상태가 함께 표시되는지 확인한다.
- 메뉴가 없는 식당에서 `메뉴 없음` 상태가 표시되는지 확인한다.
- 현재 시간이 운영 시작 전, 운영 중, 운영 종료 후인 경우를 각각 확인한다.
- 운영시간 문자열에서 시작/종료 시간을 읽을 수 있는 경우 `운영 전`, `운영 중`, `운영 종료` 상태가 올바르게 바뀌는지 확인한다.

## 네이버 캠퍼스 지도
- `메뉴 > 캠퍼스 지도` 화면을 연다.
- Google 지도가 아니라 네이버 지도가 표시되는지 확인한다.
- 초기 카메라가 한양대학교 ERICA 캠퍼스 주변으로 잡히는지 확인한다.
- 지도를 드래그하거나 확대/축소했을 때 현재 화면 영역에 맞는 건물 마커 또는 클러스터가 표시되는지 확인한다.
- 건물이 밀집한 축척에서는 클러스터 마커가 표시되는지 확인한다.
- 클러스터 마커를 누르면 해당 위치로 확대되는지 확인한다.
- 건물 또는 강의실을 검색한다.
- 검색 결과를 선택한다.
- 선택한 위치로 카메라가 이동하고 해당 건물/강의실 마커가 표시되는지 확인한다.
- 검색 결과 선택 후 지도 이동 제어가 잠기고 `위치 초기화` 버튼이 표시되는지 확인한다.
- `위치 초기화` 버튼을 누르면 캠퍼스 초기 위치로 돌아가고 지도 이동 제어가 다시 가능해지는지 확인한다.
- 지도 마커를 누른다.
- 외부 지도 앱이 열리고 선택한 위치가 표시되는지 확인한다.
- 외부 지도 앱이 없는 환경에서는 앱이 종료되지 않고 토스트가 표시되는지 확인한다.

## 네이버 셔틀 정류장 지도
- `셔틀버스` 실시간 화면을 연다.
- 상단 정류장 정보 또는 정류장 안내로 셔틀 정류장 상세 Sheet를 연다.
- Sheet 안의 지도가 네이버 지도로 표시되는지 확인한다.
- 해당 셔틀 정류장 위치에 마커가 표시되는지 확인한다.
- 초기 카메라가 정류장 주변으로 확대되어 있는지 확인한다.
- Sheet 내부 지도에서 스크롤, 확대/축소, 회전, 틸트 제스처가 막혀 있는지 확인한다.
- 정류장 첫차/막차 정보가 지도 아래에서 기존과 동일하게 표시되는지 확인한다.

## 네이버 버스 정류장 지도
- `버스` 실시간 화면을 연다.
- 노선 또는 정류장 정보에서 버스 정류장 상세 Sheet를 연다.
- Sheet 안의 지도가 네이버 지도로 표시되는지 확인한다.
- 해당 버스 정류장 위치에 마커와 정류장 이름이 표시되는지 확인한다.
- 초기 카메라가 정류장 주변으로 확대되어 있는지 확인한다.
- 첫차/막차 목록이 지도 아래에서 기존과 동일하게 표시되는지 확인한다.

## 네이버 대체노선 정류장 Sheet
- 셔틀버스 실시간 화면에서 대체노선 정보 버튼을 누른다.
- 대체노선 Sheet가 열리고 제목 아래 전체 영역에 네이버 지도가 표시되는지 확인한다.
- 셔틀버스 정류장 마커와 노선버스 정류장 마커가 서로 다른 아이콘/색상으로 표시되는지 확인한다.
- 두 정류장 정보창이 지도 위에 표시되는지 확인한다.
- 정보창을 눌러도 카메라가 임의로 이동하지 않는지 확인한다.
- 두 정류장이 얇은 실선 경로와 화살표로 연결되어 있는지 확인한다.
- 초기 카메라가 두 정류장이 모두 보이도록 잡히는지 확인한다.
- 위치 권한이 허용된 상태에서 현재 위치가 두 정류장과 함께 화면 안에 들어오도록 카메라가 조정되는지 확인한다.
- 위치 권한이 거부된 상태에서도 앱이 종료되지 않고 두 정류장 기준으로 지도가 표시되는지 확인한다.
